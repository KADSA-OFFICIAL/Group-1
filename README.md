# 방과 후 (Group-1)

2D 탑다운 공포 방탈출 게임입니다. Godot 4.6으로 만들고 있습니다.

## 게임 소개

주인공은 학교에 두고 온 **국어책**을 가지러 밤의 학교에 다시 들어갑니다. 책을 찾던 중 창문 너머로 **시체를 묻는 수위 아저씨**를 목격하고, 들키지 않으려다 학교 안에 갇히고 맙니다. 어두운 학교를 탐색하며 단서와 열쇠를 모아 **학교를 탈출하는 것**이 목표입니다.

### 핵심 특징

- **어둠과 손전등** — 학교 전체가 어둡고, 플레이어 주변 일정 반경만 보입니다. 시야 밖에 무엇이 있을지 모릅니다.
- **5층 학교 탐색** — 층마다 테마가 다른 70여 개의 방이 있습니다. 방은 문으로만 드나들 수 있습니다.
- **조사와 아이템** — 방 안의 물건을 조사해 아이템을 얻고, 잠긴 곳은 맞는 아이템이 있어야 열립니다.

## 조작법

| 키 | 동작 |
|----|------|
| `W` `A` `S` `D` 또는 방향키 | 이동 |
| `E` | 조사 / 상호작용 |
| `R` | 소지품 확인 |

## 플레이 방법

1. **시작** — 타이틀 화면에서 `게임 시작`을 누르면 짧은 도입 내레이션(E/Enter로 넘기기)이 지나간 뒤 5층 미술실에서 시작합니다. 미술실 문은 잠겨 있으니 방 안에서 열쇠를 찾아 여세요. 그 뒤 계단으로 내려가 1층 현관으로 탈출하세요.
2. **시야** — 화면은 플레이어를 따라다니고, 플레이어 주변만 밝게 보입니다. 벽과 방 윤곽은 희미하게만 보이니 천천히 움직이며 파악하세요.
3. **탐색** — 각 방의 **갈색 문 표식**이 있는 곳으로만 출입할 수 있습니다. 벽은 통과할 수 없습니다.
4. **계단** — 계단실은 가운데 난간으로 반이 나뉘고, 바닥의 **화살표 표시가 이동 방향**(위 = 위층, 아래 = 아래층)을 알려줍니다. 화살표 방향의 반쪽으로 들어가 끝까지 걸어가면 층이 이동됩니다. 화살표가 없는 반쪽은 막혀 있습니다. **계단 입구는 기본적으로 잠겨 있습니다** — 각 층 어딘가에 숨겨진 그 층의 계단 열쇠를 찾아 `E`로 여세요. 한 번 열면 그 층의 계단 두 곳이 모두 열리고 열쇠는 사용됩니다.
5. **HUD** — 화면 좌상단에 현재 **목표**와 **소지품**이 표시됩니다. `R`을 누르면 소지품 목록 창이 열립니다. 조사 결과나 안내는 화면 하단 알림으로 나타납니다. 아이템은 **최대 5개**까지만 들 수 있습니다 — 가득 차면 새 아이템을 주울 수 없습니다.
6. **조사** — 물건 가까이에서 `E`를 누르면 조사합니다. 아이템을 얻거나 메시지를 확인할 수 있습니다. 잠긴 문·물건은 필요한 아이템(예: 현관 열쇠)을 먼저 찾아야 합니다.
7. **목표** — 단서를 모아 현관을 열고 학교를 탈출하세요.

## 학교 안내

| 층 | 주요 시설 |
|----|-----------|
| 1층 | 1학년 1~3반, 교장실, 행정실, 국어·미술·음악·과학 교실, 수위실, **현관 로비(탈출 지점)** |
| 2층 | 1~7반, 교무실, 체육건강부, 체육관 입구, 샤워실, 체력단련실 |
| 3층 | 1~8반, 교무실, 인문사회부, 학생회실, 방송실 |
| 4층 | 1~8반, 교무실, 과학탐구부, 과학 실험실, 수학교구실, 역사자료실 |
| 5층 | **미술실(시작 지점)**, 시청각실, 자료실, 인쇄실, 보건실, 상담실, 방송 장비실, 동아리방, 다목적실, 옥상 창고, 옥상 계단 |

## 실행 방법

1. [Godot 4.6](https://godotengine.org/) 이상을 설치합니다.
2. 저장소를 클론하고 Godot에서 `project.godot`을 엽니다.
3. `F5`(프로젝트 실행)를 누르면 타이틀 화면부터 시작합니다.

> 층 씬을 단독 실행(F6)하면 맵만 보입니다. 플레이어·조명은 `scenes/main/main.tscn`에서 조립되므로 반드시 F5로 실행하세요.
