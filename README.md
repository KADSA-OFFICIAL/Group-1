# 방과 후 (Group-1)

2D 탑다운 공포 방탈출 게임입니다. Godot 4.6으로 만들고 있습니다.

## 게임 소개

주인공은 학교에 두고 온 **국어책**을 가지러 밤의 학교에 다시 들어갑니다. 책을 찾던 중 창문 너머로 **시체를 묻는 수위 아저씨**를 목격하고, 들키지 않으려다 학교 안에 갇히고 맙니다. 어두운 학교를 탐색하며 단서와 열쇠를 모아 **학교를 탈출하는 것**이 목표입니다.

### 핵심 특징

- **어둠과 손전등** — 학교 전체가 어둡고, 플레이어 주변 일정 반경만 보입니다. 시야 밖에 무엇이 있을지 모릅니다.
- **5층 학교 탐색** — 층마다 테마가 다른 50여 개의 방이 있습니다. 방은 문으로만 드나들 수 있습니다.
- **조사와 아이템** — 방 안의 물건을 조사해 아이템을 얻고, 잠긴 곳은 맞는 아이템이 있어야 열립니다.

## 조작법

| 키 | 동작 |
|----|------|
| `W` `A` `S` `D` 또는 방향키 | 이동 |
| `E` | 조사 / 상호작용 |

## 플레이 방법

1. **시작** — 타이틀 화면에서 `게임 시작`을 누르면 3층 복도에서 시작합니다.
2. **시야** — 화면은 플레이어를 따라다니고, 플레이어 주변만 밝게 보입니다. 벽과 방 윤곽은 희미하게만 보이니 천천히 움직이며 파악하세요.
3. **탐색** — 각 방의 **갈색 문 표식**이 있는 곳으로만 출입할 수 있습니다. 벽은 통과할 수 없습니다.
4. **계단** — 발판 무늬가 있는 계단실은 문 없이 트여 있습니다. 층 이동 지점입니다.
5. **HUD** — 화면 좌상단에 현재 **목표**와 **소지품**이 표시됩니다. 조사 결과나 안내는 화면 하단 알림으로 나타납니다.
6. **조사** — 물건 가까이에서 `E`를 누르면 조사합니다. 아이템을 얻거나 메시지를 확인할 수 있습니다. 잠긴 문·물건은 필요한 아이템(예: 현관 열쇠)을 먼저 찾아야 합니다.
7. **목표** — 단서를 모아 현관을 열고 학교를 탈출하세요.

## 실행 방법

1. [Godot 4.6](https://godotengine.org/) 이상을 설치합니다.
2. 저장소를 클론하고 Godot에서 `project.godot`을 엽니다.
3. `F5`(프로젝트 실행)를 누르면 타이틀 화면부터 시작합니다.

> 층 씬을 단독 실행(F6)하면 맵만 보입니다. 플레이어·조명은 `scenes/main/main.tscn`에서 조립되므로 반드시 F5로 실행하세요.

## 프로젝트 구조

- 배경/맵: `scenes/background` (층별 씬)
- 플레이어: `scenes/player`
- UI: `scenes/ui` (타이틀, HUD)
- 게임 상태/조립: `scenes/main`, `scripts/game`

자세한 규칙은 `docs/scene_structure.md`를 참고하세요.

## 협업 워크플로우

2인 협업은 `feat/fix/issue → dev → main` 흐름을 따릅니다. 검증된 코드만 `main`에 들어갑니다.

- 2인 협업 가이드: `docs/collaboration.md`
- 명령 흐름(스크립트): `docs/github-workflow.md`
- 작업 규칙(하네스): `CLAUDE.md`, `AGENTS.md`

```bash
scripts/start-task.sh 12 feat player-jump   # 이슈 12로 dev에서 작업 브랜치 시작
scripts/finish-task.sh 12 "Add player jump" # 커밋·푸시·dev 대상 PR
scripts/promote-main.sh 12 "Add player jump"# dev 검증 후 main 대상 PR
```
